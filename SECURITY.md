# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| all     | :white_check_mark: |

## Reporting a Vulnerability

To report a vulnerability, you are invited to open an issue in the project so that we can track the discussion.
You also can consider sending me an email, ideally signed using GPG, if you need to share privately some sensible informations.
